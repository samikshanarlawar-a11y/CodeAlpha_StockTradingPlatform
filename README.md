# CodeAlpha Stock Trading Platform

## Overview

The CodeAlpha Stock Trading Platform is a Java-based console application that simulates a simple stock trading environment. Users can view stock market data, buy and sell stocks, and track their portfolio performance.

## Features

* Display available stocks and market prices
* Buy stocks
* Sell stocks
* Track portfolio holdings
* Calculate total portfolio value
* Console-based interactive menu
* Object-Oriented Programming (OOP) implementation

## Technologies Used

* Java
* OOP Concepts
* HashMap
* Scanner Class

## Project Structure

* `Stock` Class → Stores stock details
* `User` Class → Manages user balance and portfolio
* `CodeAlpha_StockTradingPlatform` → Main class containing program execution

## How to Run

1. Open the project in:

   * VS Code
   * Eclipse
   * IntelliJ IDEA

2. Save the file as:

```text
CodeAlpha_StockTradingPlatform.java
```

3. Compile the program:

```bash
javac CodeAlpha_StockTradingPlatform.java
```

4. Run the program:

```bash
java CodeAlpha_StockTradingPlatform
```
